# Samiksha Love

A simple static project for the `samiksha-love` page.

## Files

- `index.html` - main page
- `package.json` - helper scripts to preview locally

## Run locally

Open `index.html` directly in a browser, or start a local server:

```bash
npm run dev
```

Then open:

```text
http://localhost:4173
```

## Notes

- The page uses Google Fonts, YouTube, and Google Maps embeds, so internet access is needed for those parts.
- There is a placeholder section in the page for a custom video montage if you want to add one later.
